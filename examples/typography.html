Headers

Headers can also be set by adding one of the classes .h1-.h6 if you don't want to use tags (.e.g. in SEO pages)

By default headers are scaled from $base-font-size:15px; by $ratio:1.1618 and have $header-color:text('dark-hint') as their color

On this site you can see that headers are used along with .bg-pink and .bg-pink-300 classes
